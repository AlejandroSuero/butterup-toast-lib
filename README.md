# Installation

Install this library using your node package manager.

- With **npm**

```bash
npm install butterup-toast-lib
```

- With **pnpm**

```bash
pnpm add butterup-toast-lib
```

- With **bun**

```bash
bun add butterup-toast-lib
```

- With **yarn**

```bash
yarn add butterup-toast-lib
```
